"cd" : entra em um diretorio
"cd.." : sai de um diretorio
"mkdir": cria um diretorio
	
"echo": printa o valor de um arquivo na tela
	"echo 'String' > ''arquivo'' " substitui os dados do arquivo em questão pelo valor decidido
	"echo 'String >> ''aquivo'' " adiciona o valor no arquivo em questão
	
"ls": lista os diretorios e arquivso dentro do diretorio atual
"touch": cria um arquivo 
"mr" : remove o arquivo
"mv" : renome aquivos
"git status" : mosta os status do git
"nano": abre um arquivo no editor nano
"git init": inicia o repositorio git
"git remote -v": printa o repositorio conectado
"git remote add":  conecta no repositorio
"git push -u origin main": envia as alterações
"git add 'arquivo'":  prepara o arquivo para o commit
